# git使用


使用步骤
---

1. 安装git
2. 设置全局的 name、email
3. 生成本地SSH key
4. github 上添加SSHkey
5. 新建本地仓库
6. github上新建远程仓库
7. 将本地仓库和远程仓库关联起来
8. 第一次提交
9. 后续提交


分支操作
---
1. 查看分支  git  branch
2. 新建分支  git  branch  分知名
3. 切换分支  git  checkout 分知名
4. 新建+切换分支  git  checkout -b  分知名
5. 合并某分支到当前分支  git merge  分支名
6. 删除分支  git branch -d  分知名
